# 🚦 Semáforo

Simulador interativo de semáforo com HTML, CSS e JavaScript.

## 📋 Como Usar

1. Abra `index.html` no navegador
2. Clique em **"Iniciar"** para começar a simulação
3. Clique em **"Parar"** para interromper

## 🎨 Funcionamento

O semáforo cicla automaticamente:
- 🔴 **Vermelho** - 2 segundos
- 🟢 **Verde** - 2 segundos  
- 🟡 **Amarelo** - 2 segundos

## 📁 Arquivos

- `index.html` - Estrutura
- `style.css` - Estilos
- `script.js` - Lógica

## 💡 Conceitos Utilizados

- **Promises & async/await** - Gerenciar intervalos de tempo
- **Event Listeners** - Capturar cliques dos botões
- **DOM Manipulation** - Controlar cores dinamicamente

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript ES6+
